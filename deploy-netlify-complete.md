# Deploy to Netlify - Complete Guide

## Step 1: Go to Netlify
1. Visit: https://app.netlify.com/drop
2. Sign up/login with GitHub/GitLab/Email

## Step 2: Drag & Drop Deployment
1. Drag your entire `d:\travi` folder into the drop zone
2. Wait for deployment (2-3 minutes)
3. Get your free URL: `https://your-project-name.netlify.app`

## Step 3: Connect to GitHub (Recommended)
1. In Netlify dashboard → "New site from Git"
2. Choose GitHub
3. Select `the-pride-holidays` repository
4. Build settings: Use `netlify.toml` (auto-detected)
5. Click "Deploy site"
