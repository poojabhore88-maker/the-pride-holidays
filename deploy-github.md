# Deploy to GitHub Pages (Free)

## Step 1: Create GitHub Repository
1. Go to https://github.com
2. Click "+" → "New repository"
3. Name: `unexia-travels`
4. Description: "The Pr♦de Holidays - Domestic Travel Website"
5. Make it Public
6. Click "Create repository"

## Step 2: Upload Files
1. Click "uploading an existing file"
2. Drag all files from `d:\travi` folder
3. Include: index.html, destinations.html, services.html, about.html, contact.html, styles.css, script.js, assets folder
4. Commit changes

## Step 3: Enable GitHub Pages
1. Go to repository → Settings → Pages
2. Source: "Deploy from a branch"
3. Branch: "main" → "/ (root)"
4. Click "Save"

## Step 4: Get Your URL
Wait 2-3 minutes, then visit:
`https://yourusername.github.io/unexia-travels`

## Benefits:
- ✅ Completely free
- ✅ HTTPS security
- ✅ GitHub integration
- ✅ Version control
- ✅ Easy updates

## Note: Don't upload sensitive files like:
- README.md with setup instructions
- deploy-*.md files
- Any configuration files with API keys
